# FocusGuard

공부에 방해되는 웹사이트 사용을 줄여주는 미니멀 Chrome Extension입니다.

---

## ✨ Features

* 🎯 집중 모드 ON/OFF
* 🚫 차단 사이트 추가 및 삭제
* ⏱️ 사이트별 10분 사용 제한
* 🛑 제한 시간 초과 시 자동 차단 페이지 이동
* 💾 설정값 Chrome Storage 저장

---

## 🛠️ Tech Stack

* React
* TypeScript
* Vite
* Chrome Extension Manifest V3
* Chrome Storage API

---

## 🚀 Usage

* (배포 후 추가예정)

---

## 🔒 Privacy

FocusGuard는 개인정보를 수집하지 않습니다.  
차단 사이트와 집중 모드 설정은 모두 브라우저 내부의 `chrome.storage.local`에만 저장됩니다.
